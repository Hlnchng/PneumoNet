# PneumoNet

Pneumonia image classification using CNN for educational purposes.

### 0.1.0 - `vanilla_cnn`
A basic CNN implemented from scratch to classify radiology images.

### 0.2.0 - `resnet101`
Uses the `ResNet-101` architecture to classify radiology images.
